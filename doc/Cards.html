    Cards (cards v0.2.0)

    Provides methods for creating and handling a deck of cards

    Summary

    Functions

    contains?(deck, card)

    Checks if a deck contains a specific card

    create_deck()

    Returns a list of strings representing a deck of playing cards

    create_hand(hand_size)

    Creates a hand of a given size The hand_size argument indicates how many cards should be in the hand.

    deal(deck, hand_size)

    Divides a deck into a hand and the remainder of the deck. The hand_size argument indicates how many cards should be in the hand. Returns a tuple of {hand, remainder_of_deck}.

    load(filename)

    Loads a deck from a file

    save(deck, filename)

    Saves a deck to a file

    shuffle(deck)

    Shuffles a deck of cards
